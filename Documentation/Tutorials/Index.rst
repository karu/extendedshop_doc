Tutorials
---------

First of all install the dependencies: toi\_category, tt\_address,
static\_info\_tables. We suggest to install also
sr\_feuser\_registration to manage your customers identities. If you
want to use ajax functionalities xajax extension is required.

Now you can import and install Extended Shop.
